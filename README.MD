# Crud movies 🎬
***
Project developed in Angular. It is a crud that contains information about movies and their descriptions

## Technologies 🛠️
* **Angular**
* **Mysql2**
* **Bootstrap**
* **Datatables**
* **Express**
* **Nodemon**
* **Morgan**

***

## Installation 📝
***

$ git clone git@github.com:davidB2ya/Video-Club-CRUD-Angular.git
$ cd ../web/crud-movie/
$ ng serve --open

### Pre-requisitos 📋

_Solo se requiere de un navegador_
